# Emoji Mosaic

将图像转换为由表情符号制成的马赛克。

[Live site](http://Rodneylzy.github.io)

# 使用方法
1.点击"选择文件"并上传你的照片
2.转换速度受运行平台性能以及图片大小限制，大型图片需要比较长的处理时间
3.转换完成后，"Save as image"选项将变得可用，单击并保存输出图片即可
